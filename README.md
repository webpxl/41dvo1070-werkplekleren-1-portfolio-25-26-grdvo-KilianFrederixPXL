# Portfolio - Kilian Frederix

## Over dit project

Dit is mijn persoonlijke portfolio website voor het vak **Werkplekleren 1 (WPL1)** binnen de opleiding **Digitale Vormgeving** aan de PXL Hogeschool.

De website toont mijn vaardigheden als developer en designer en bevat projecten, skills, werkervaring en een reflectie op mijn leerproces.

## Design Keuzes

### Kleurenpalet
```css
--bg-color: #EBE6C8        /* Lichte achtergrond */
--primary-color: #478c9a   /* Primaire tint (blauw/groen) */
--secondary-color: #D7A24D /* Secundaire tint (goud) */
--accent-color: #CC403B    /* Accent kleur (rood) */
--cards-color: #E2CD96     /* Card achtergrond */
--border-color: #141615    /* Donkere borders */
```

### Typografie
- **Headings:** Titan One (Google Fonts) - Bold, karaktervol display font
- **Body text:** Merriweather Sans (Google Fonts) - Leesbaar sans-serif font

### Design Principes
- **Responsive-first:** Mobiel eerst, dan tablet en desktop
- **Toegankelijkheid:** Semantische HTML, ARIA labels, voldoende contrast
- **Consistentie:** Herhaling van stijlelementen door hele website
- **White space:** Ruimte om content te laten ademen

## Deployment

Deze website wordt gehost via:
- [ ] **GitHub Pages**
- [x] **Netlify** - [https://kilianfrederixdvoportfolio.netlify.app/](https://kilianfrederixdvoportfolio.netlify.app/)

### Deployment Instructies:

#### GitHub Pages:
1. Ga naar repository settings
2. Scroll naar "Pages" sectie
3. Selecteer "main" branch als source
4. Sla op en wacht op deployment
5. Website is beschikbaar op: `https://username.github.io/repo-name/`

#### Netlify:
1. Log in op [Netlify](https://www.netlify.com/)
2. "Add new site" → "Import an existing project"
3. Verbind met GitHub repository
4. Deploy instellingen: Build command: (leeg), Publish directory: `/`
5. Deploy!
