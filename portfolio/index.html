Eindreflectie WPL1

AIM-reflectie op voorbije periode

De AIM-methode helpt om gestructureerd te reflecteren op ervaringen. AIM staat voor: Action (wat heb ik gedaan?), Impact (wat was het effect?) en Meaning (wat betekent dit voor mij?).

A Action - Wat heb ik gedaan?

[Beschrijf hier concreet welke activiteiten je hebt ondernomen tijdens WPL1. Denk aan: projecten, opdrachten, samenwerkingen, leermomenten, uitdagingen, etc.]

Voorbeeld: "Tijdens WPL1 heb ik gewerkt aan verschillende webprojecten waarbij ik HTML, CSS en JavaScript heb toegepast. Ik heb mijn eigen portfolio opgezet waarbij ik aandacht heb besteed aan responsive design..."

I Impact - Wat was het effect?

[Beschrijf hier wat het effect was van je acties. Wat heb je bereikt? Wat heb je geleerd? Hoe hebben anderen gereageerd op je werk?]

Voorbeeld: "Door deze projecten heb ik mijn technische vaardigheden sterk verbeterd. Ik merk dat ik zelfstandiger problemen kan oplossen en dat ik meer vertrouwen heb gekregen in mijn kunnen als developer..."

M Meaning - Wat betekent dit voor mij?

[Beschrijf hier wat deze ervaringen voor jou betekenen. Wat neem je hiervan mee? Hoe past dit in je ontwikkeling als professional?]

[Beschrijf hier wat jou uniek maakt. Denk aan:
